Information:


    A smaller practise app with incrementing or reducing a number using Redux actions.


    Using html5, css3, React and Redux
    


Instructions:


    1. Inside the root project directory run ' npm install ' to install the necessary packages (displayed inside package.json)

    2. run ' npm start ' and wait until it launches a server, then wait until its opened in your browser (or go into ' http://localhost:3000/  ' yourself)
